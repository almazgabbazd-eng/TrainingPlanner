# GitHub User Finder

**Автор:** Алмаз Габбазов

## Описание

Приложение для поиска пользователей GitHub с возможностью сохранения избранных пользователей.

**Основные функции:**
* Поиск пользователей GitHub по имени
* Отображение результатов в табличном виде
* Добавление пользователей в избранное
* Сохранение избранных пользователей в JSON-файл
* Проверка корректности ввода

## Технические детали

* **Язык:** Python 3.x
* **GUI:** tkinter
* **HTTP-запросы:** requests
* **Хранение данных:** JSON
* **API:** GitHub REST API

## Как использовать API GitHub

Приложение использует публичный API GitHub без аутентификации. Ограничения:
* До 60 запросов в час с одного IP (без аутентификации)
* Базовый URL: `https://api.github.com`
* Endpoint для поиска: `/search/users`
